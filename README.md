# Car Rental System (Java Console Application)

Explore the world of car rentals with the Car Rental System, a Java-based console application designed to simulate real-world car rental operations while strengthening core Java concepts. 
This project deals with hands-on experience with object-oriented programming, collections, and console-based interaction.

 ## Features

### Rent a Car
Rent available cars through an interactive console menu.
###  Return a Car
Return rented cars and automatically update availability status.
### Customer Management
Add new customers and store customer details.
### Car Management
Manage cars with brand, model, and pricing information.
### Rental History
Track rental records including customer details, rented car, and duration.

### Technologies Used

Java

Object-Oriented Programming (OOP)

Collections Framework

Console-Based User Interface
